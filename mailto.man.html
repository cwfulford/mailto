NAME

mailto - wrapper script to invoke mutt from firefox mailto: URIs.

SYNOPSIS

mailto [ -c <config_file> ] [ -d ] [ -v ]

AVAILABILITY

mailto is a Bourne shell script which should work equally well on all versions of UNIX, Linux and Mac OS X.

DESCRIPTION

mailto passes the parameters from the mailto: URI to a virtual terminal running mutt.

OPTIONS

-c    Use the specified configuration file in preference to the default "/usr/local/etc/mailto.cf" . If no configuration file is found the script will continue with the default MUA mutt running in terminal ""

-d    Set debug mode on. The -x    option is set within the shell and each command in the script is echoed to the console.

-v    Doesn't do anything at all at present and probably never will. report.

FILES

mailto.cf.

EXAMPLES

	/usr/local/bin/mailto

In firefox prefences, applications, mailto, Use other ..., just type in the full path name to the script as above.

BUGS

The script is quite crude and has very little, if any, error checking. The consequences of unexpected input are untested.
